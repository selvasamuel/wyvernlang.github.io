Contributing

The Wyvern source code can be found here.

To run the tests, first, clone the repo. Set your a variable that references the build directory, then change into it. Then, run ant. 


                                git clone git@github.com:wyvernlang/wyvern.git $HOME/wyvern
                                WYVERN=$HOME/wyvern
                                cd $WYVERN/tools
                                ant test

Don't have ant? You can download it here.
